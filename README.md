# Lucky 7 

A simple betting game made with Python.

Players bet on whether the sum of two dice will be:

- Above 7
- Below 7
- Exactly 7

If the prediction is correct, the player wins 2x or 3x their money depending on what their bet was.

## How to Run

```bash
python lucky7.py
````

## Built With

* Python
* random module
